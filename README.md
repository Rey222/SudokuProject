# Sudoku Project

Простая мобильная игра **Судоку** для Android, разрабатываемая в учебных целях.  
Проект находится на ранней стадии разработки и постепенно расширяется новыми функциями.

---

## Основные функции (в плане)

- Выбор уровня сложности: **Легкий**, **Средний**, **Сложный**
- Интерактивная игровая доска 9×9
- Ввод чисел через кнопки
- Подсчёт ошибок (максимум 3 жизни ❤️)
- Проверка корректности решения
- Экран победы / поражения
- Правила игры в отдельном диалоговом окне

> ⚠️ На текущем этапе реализованы только экраны меню и основной активити без игровой логики.

---

## Технологии

- **Язык**: Kotlin
- **IDE**: Android Studio
- **Архитектура**: Activity + View
- **UI**: ConstraintLayout, Button, Spinner, AlertDialog
